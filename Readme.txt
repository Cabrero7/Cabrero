Descripción
Este proyecto es un ejemplo práctico de un documento HTML que demuestra varias características fundamentales del desarrollo front-end utilizando HTML y CSS. El archivo incluye ejemplos de estructura básica, encabezados, divisiones, enlaces, listas, elementos multimedia, tablas y formularios.

Estructura del Proyecto
El proyecto se compone de un solo archivo HTML que incluye:

Declaración DOCTYPE: Define la versión del HTML utilizada.
Metadatos: Incluye charset y viewport para asegurar compatibilidad y adaptación en dispositivos móviles.
Estilos CSS: Se incluye un estilo CSS básico directamente en la etiqueta <style> para manejar los bordes de tablas y las propiedades de header y footer.
Encabezado: Un header estilizado con un título centrado.
Main Content: El contenido principal está dividido en secciones que abordan diferentes conceptos y elementos del HTML.
Características de las Secciones
1. Estructura Básica del Documento HTML
Sección que describe la estructura mínima necesaria para un documento HTML válido, con metadatos y un título.

2. Encabezados y Propiedades del Texto
Muestra diferentes estilos de texto, incluyendo negrita, cursiva y subrayado.
3. Divisiones y Agrupación de Contenido
Utiliza etiquetas <div> para agrupar contenido en secciones.
4. Enlaces y Listas
Incluye ejemplos de enlaces externos e internos, listas ordenadas y desordenadas.
5. Elementos Multimedia
Demuestra la inclusión de elementos multimedia como imágenes, videos y audios con controles.
6. Tablas
Sección que muestra la construcción de una tabla simple con encabezados y datos.
7. Formularios
Incluye un formulario básico con campos para nombre, correo electrónico y un área de texto para mensajes.

Ejecución
Para visualizar este proyecto:
Guarda el contenido del código en un archivo con la extensión .html, por ejemplo, practico_frontend.html.
Abre el archivo con cualquier navegador web.
